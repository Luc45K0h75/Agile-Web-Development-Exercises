Kidney Stone CT Scan Image Detector

Project Status

Completion Status: Almost completed, Estimated Finish: June/July 2026

Project Aim

The goal of this project is to create a convolution neural network (CNN) without a pretrained base that can accurately detect whether or not a kidney stone exists in a CT scan.

Project Dataset

The dataset I am using is the Axial CT Imaging Dataset (Kidney Stone Detection) by Abdalla et al. that contains a total of 38,821 images, 3364 of which are original images of CT scans sourced from medical centers in Sulaimani and Rania, Kurdistan Region, Iraq as well as 35,457 additional augmented images. (Abdalla et al., 2025) I chose this dataset due to the large number images which can help the model to recognise clear patterns amongst scans. This helps in reducing underfitting as it allows for the indentification of common patterns which helps minimise noise and increase common feature recognition. Having a large variety of variable images helps the model get used to many different and variable features that can be used to detect images in Kidney Stones which in turn helps to counter overfitting. Additionally, the presence of augmented images helps decrease overfitting as it makes feature recognition better through exposing the model to altered versions of the same image. This makes it harder for the model to memorise and makes it better at generalising. (Holbrook, Cook, 2023).
